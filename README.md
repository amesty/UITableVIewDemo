# UITableVIewDemo
一个关于tableview的总结，其中包含等高cell（仿团购）和非等高cell（仿微博）的实现思路，tableview强大的编辑模式，删除，添加，移动，批量操作，自定义左滑标题，仿照QQ好友分组功能，以及消息聊天界面的简单搭建！ 基本包括大部分常用的tableview原生自带功能！
